### 4. 相机控制增强
- **状态**: ✅ 已完成
- **完成时间**: 2026-05-04
- **描述**:
  - **Alt + 左键拖动**：旋转视角（任何时候）
  - **Ctrl + 左键拖动**：缩放视角（任何时候）
  - **Shift + 左键拖动**：平移视角（任何时候）
  - **按下 Ctrl+H**：回到默认视图（棋盘面向视口，俯视角度）
  - **修饰键按下时**：不处理棋子移动，优先响应相机操作
  - **无修饰键时**：正常拖动棋子
