- 使用 `std::fs::read_dir` 遍历目录
- 使用 `std::fs::read_to_string` 读取文件内容
- 多路径兼容策略确保在不同环境下都能找到资源

### 2. Tauri IPC 通信
- 前端通过 `invoke` 调用 Rust 命令
- 异步 Promise 处理
- 错误信息透传到前端

- Composition API (`<script setup>`)

## 🔧 已知问题与改进建议

### 当前限制
1. 仅支持 `.txt` 格式的棋谱文件
2. 棋谱文件必须放在 `src/assets/chess_score` 目录下
3. 不支持棋谱文件的删除、重命名等管理操作

### 未来优化方向
